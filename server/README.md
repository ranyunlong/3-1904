## 安装
```shell
npm i express
```

## express的项目生成工具 cli

```shell
npm install express-generator -g
```

## 使用node-dev 热更新代码

```shell
npm i -g node-dev
# 当有代码发生改变 他会自动重启服务器
```

需要用node-dev 来启动你的项目

```shell
node index.js
# 替换成
node-dev index.js
```
